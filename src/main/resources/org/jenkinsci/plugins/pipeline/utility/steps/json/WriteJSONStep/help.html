Write a JSON file in the current working directory and/or have the JSON returned as a string. That for example was previously read by readJSON.

Fields:
  • json: The JSON object to write.
  • file (optional): Path to a file in the workspace to write to. Must either have a file or return encoded JSON with returnString
  • returnString (optional): Set to *true* to return JSON as string. Can be used alone or with file
  • pretty (optional): Prettify the output with this number of spaces added to each level of indentation.

Example:

def input = readJSON file: 'myfile.json'
//Do some manipulation
writeJSON file: 'output.json', json: input
// or pretty print it, indented with a configurable number of spaces
writeJSON file: 'output.json', json: input, pretty: 4
// or get text back instead of writing a file
writeJSON returnString: true, json: input
// or get text back AND write a file
writeJSON file: 'output.json', returnString: true, json: input
